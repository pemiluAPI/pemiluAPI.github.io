Apa itu API Pemilu?

Subscribe Untuk Update

Para developer menggunakan API Pemilu untuk mengembangkan aplikasi mobile dan web; temukan aplikasi-aplikasi tersebut di pemiluapps.org dan cobalah.

Bagaimana cara mengembangkan aplikasi menggunakan API Pemilu

Jika kamu ingin mengembangkan aplikasi dengan API ini, silahkan kunjungi situs khusus developer di: http://developer.pemiluapi.org/

Kamu bisa mendaftar untuk mendapatkan key API di sana, atau gunakan key gratis ini: fea6f7d9ec0b31e256a673114792cb17

Kami sarankan agar anda mendaftarkan key anda sendiri apabila akan menggunakan API ini untuk aplikasi produksi.

Dokumentasi untuk endpoint API dapat dilihat di sini: http://developer.pemiluapi.org/endpoints

Bagaimana cara berkontribusi untuk API Pemilu

API Pemilu sepenuhnya adalah proyek sumber terbuka (open-source). Kamu bisa menemukan semua kode untuk API ini pada laman GitHub organisasi kami. Kamu juga bisa menemukan dan mengunduh semua raw data dari API ini di repository pemilu-data di sana.

Jika kamu ingin berkontribusi, carilah proyek yang sesuai dengan minatmu, fork the code, lalu submit "pull request". Kembangkan aplikasimu sendiri dengan menggunakan API ini dan beritahu kami hasilnya, seperti yang telah dilakukan oleh developer lain. Kamu juga bisa mencari open issues pada repository manapun dan bantu kami memperbaikinya. Atau mulai issues baru mengenai repository manapun bila kamu menemukan bugs, kesalahan, atau peluang untuk perbaikan. API ini dapat menjadi lebih baik apabila didukung oleh komunitas developer dan orang-orang yang berkomitmen dan berdedikasi agar API ini selalu up-to-date, akurat dan berguna.

Yang saat ini tersedia di API Pemilu

Seluruh data yang disediakan API ini dapat diunduh dari repository pemilu-data di GitHub.

Caleg

Melalui endpoint API Caleg, kamu bisa mendapatkan informasi mengenai seluruh kandidat DPR dan DPD (ada 7.552 calon!) termasuk foto dan informasi biografis mereka, serta lebih dari 20.000 calon DPRD I.

Geografis

Endpoint API Geografis menyediakan daftar calon berdasarkan lokasi, kamu juga daoat mengakses peta terkini pada level provinsi, kabupaten/kota, dan kecamatan.

Hukum & Peraturan dan FAQ Pemilu

Melalui endpoint API Hukum & Peraturan dan FAQ Pemilu, kamu dapat menemukan jawaban atas pertanyaan umum tentang pemilu yang dilengkapi dengan referensi dan kutipan peraturan acuan.

Incumbent

Dengan endpoint API Incumbent, kamu dapat lebih mengenali anggota-anggota DPD dan DPR saat ini.

Berita & Informasi

Dan dengan endpoint API Berita & Informasi, kamu bisa mendapatkan berita pilkada terbaru yang relevan dengan kepentingan mereka.

Dan lebih banyak lagi

Bila kamu menggunakan API Pemilu, kamu akan mendapatkan informasi yang bebas, terbuka, terkini dan terverifikasi tentang pemilu dan pemerintah. Buatlah sesuatu yang menarik dan beritahu kami!
