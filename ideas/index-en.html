ELECTIONS API 2014

bahasa - english

Below are some ideas for apps that can be built on the ELECTIONS API 2014, an API for the 2014 Indonesian elections.

If you're interested in building an app or working on the project in another way, send an e-mail to contact@pemiluapi.org or tweet @PemiluAPI.

You can find the beginnings of a technical proposal for the API in this site's README on GitHub.

If you're interested in talking about the project, you can start or join a conversation on this site's GitHub Issues page.

Or you can take this short survey and tell us how we can reach more people with the election information they need most.
